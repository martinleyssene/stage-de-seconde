# Mon deuxième cerveau

Petite application de prise de notes réalisée pendant un **stage de seconde**.
Tout tient dans un seul fichier HTML : pas d'installation, pas de serveur.
Tu double-cliques sur le fichier et il s'ouvre dans ton navigateur.

## Comment l'utiliser

1. Ouvre `Mon deuxième cerveau.html` dans un navigateur (Chrome, Firefox, Safari…).
2. Écris un **titre** et un **contenu**, puis clique sur **Ajouter la note**.
3. Déplace les notes à la souris pour organiser tes idées.

## Fonctionnalités

- **Créer des notes** : une idée par note, avec un titre et un contenu.
- **Modifier / supprimer** une note (boutons ✏️ et ❌).
- **Déplacer** les notes librement dans la zone de travail.
- **Relier les notes** entre elles (bouton 🔗) pour créer des liens, et les supprimer (bouton 🗑️).
- **Sauvegarde automatique** dans le navigateur (les notes restent après fermeture).
- **Télécharger** tes notes dans un fichier `.json`, et les **recharger** plus tard (📤).

## Comment c'est fait

Le fichier mélange les trois langages du web, le tout abondamment commenté en français :

- **HTML** : le squelette (boutons, zones de texte).
- **CSS** : la déco (couleurs, tailles).
- **JavaScript** : le cerveau (ce qui se passe quand on clique).

## Notes pour qui débute

Pas besoin de tout comprendre d'un coup : les commentaires du fichier
expliquent chaque partie pas à pas. N'hésite pas à modifier les couleurs
dans la section CSS pour t'entraîner.
